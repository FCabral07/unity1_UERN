Esse jogo foi criado usando toda a lógica de programação em C, usei apenas para a interface gráfica, em algumas partes, 
o C++. Fiz isso pois esse é um trabalho para a universidade em que, um dos requisitos, era que o jogo 
fosse criado em C. Essa é apenas uma versão de um iniciante em programação, espero que gostem.

O jogo é o tradicional jogo da velha, caso você faça uma sequência(de 3) do X ou O, seja na diagonal, vertical 
ou horizontal, você ganha, caso contrário o jogo é declarado empate (mais conhecido no Brasil como "velha").

//EN-US

This game was created using all the programming logic in C, I used only for the graphical interface, in some parts, 
C++. I did this because this is a job for the university where one of the requirements was that the game 
be created in C. This is just a beginner's version of programming, I hope you like it.

The game is the traditional tic-tac-toe, if you make a sequence(3) of the X or O, either diagonally, vertically
or horizontal, you win, otherwise the game is declared a draw (better known in Brazil as "old").
